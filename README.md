# Calculateur Pension Alimentaire Belgique

Calculateur de pension alimentaire belge basé sur la **méthode Renard** — la méthode de référence utilisée par les tribunaux belges depuis 2016.

Calcule automatiquement la contribution de chaque parent selon les revenus nets et l'indice Statbel 2026.

**Site :** [calculateur-pension-alimentaire.be](https://calculateur-pension-alimentaire.be)
